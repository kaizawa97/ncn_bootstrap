What is NCN?

NCNは、1)Recruitingと2)Knowledgeの2つを軸に、
様々な活動に従事している非法人団体である。

Recruting

Ex)世界トップレベル大学/卒業生を採用する為の
採用戦略策定～実行支援

Knowledge

Ex)日本に対する知見を深める事を目的とした
ツアー / ワークショップ開催

諸活動を通じて、日本に対する世界トップレベル学生/卒業生の知見や、
日本への関心を集め、日本市場のグローバル化と成長を両立させる事を目的としている。
NCNは、「多様性を受け入れる土壌を日本に醸成する」をビジョンに掲げている。
ビジョン実現の為に、「多様性を尊重する文化の醸成を通じた、日本経済の成長実現」を
ミッションと定義している。

Vision

日本に住む人々が、お互いの違いを
尊重し合いながら生活できる文化の醸成

Mission

多様性を尊重する文化の醸成を通じた、
日本経済の成長実現

Recruiting

NCNは、日系企業と世界トップレベルの学生が密接な接点を持つ機会を提供する為に、
毎年ハーバード大学をはじめとした世界トップレベルの大学で日系企業と提携しキャリアフェアの開催を行っている。
